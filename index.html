Resource Hub

Front end application integrating data via google sheets api to compile helpful resources pertaining to modern development practices.

Pineapple Kingdom

Collaborative roleplaying project with character creation and login interface based on the Choose Your Own Adventure series of books. The back end was built using a Mongo database deployed on MongoDB Atlas interfacing with a custom API using express.js which was deployed via Heroku. The front end styling was accomplished with CSS, HTML, JavaScript, and JSX in a custom React application and deployed with GitHub pages.

Study Buddy

Implemented a learning style assessment which matches the student to one of 3 learning styles. They are then directed to helpful resources to encourage efficient study habits that align with their learning style. This front end application was developed with React, JavaScript, HTML, and CSS. Version control was accomplished with Git and the site was deployed with GitHub pages.
